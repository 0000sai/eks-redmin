# Redmine AWS EKS Setup

# Prerequisites

1. ACM Setup with your domain

2. packages Installed in provided packages.sh

3. Docker (Optional)

4. VsCode (Optional)

# Steps

1. cd eks/environment/dev

2. aws configure 

3. terraform apply

4. Resources Provisioned VPC, RDS, KMS, AWS Secrets,EKS CLUSTER, EKS Manged Nodes

5. Terraform Destroy
